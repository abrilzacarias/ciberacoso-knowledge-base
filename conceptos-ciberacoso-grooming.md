# Conceptos de Ciberacoso y Grooming (Argentina)

## Frase orientativa
“El celular de tu hijo no debería recibir lo que la ley prohíbe.”

## Ciberacoso (definición general)
El ciberacoso es la intimidación, el hostigamiento o la humillación mediante el uso de tecnologías digitales.  
Puede darse en:

- Redes sociales  
- Plataformas de mensajería  
- Plataformas de juegos  
- Teléfonos móviles  

El comportamiento suele ser **repetido** y tiene el objetivo de **atemorizar, enfadar o humillar**.

### Ejemplos frecuentes de ciberacoso
- Difusión de mentiras, fotos o videos vergonzosos.  
- Envío de mensajes hirientes, abusivos o amenazantes.  
- Suplantación de identidad para enviar mensajes ofensivos.  
- Uso de IA generativa para producir contenido sexualizado o intimidatorio sin consentimiento.

---

## Grooming (definición general)
El grooming ocurre cuando un adulto establece contacto con un menor mediante tecnologías digitales con el objetivo de vulnerar su integridad sexual.

En Argentina, esta conducta está contemplada en la legislación vigente.  
El Código Penal incluye disposiciones orientadas a este tipo de situaciones (por ejemplo, el contacto con un menor con fines de abuso sexual), pero esta base de conocimiento no sustituye la lectura de normas oficiales ni asesoramiento especializado.
